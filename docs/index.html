OnTrack

A aplicação OnTrack foi desenvolvida no ambito da unidade curricular Engenharia de Software, do ano 4 do Mestrado Integrado em Engenharia de Computadores e Telemática, no ano letivo 2020-2021.

Descrição

Se procura perceber melhor o conceito deste projeto, tal como as features que ele oferece e as tecnologias que utiliza, está no sítio certo. Saiba mais!

Visão

O objetivo do OnTrack é proporcionar uma sistema simples e interativo, que serve não só os utilizadores dos autocarros do Porto como também o administrador/gestor do sistema.

Especificações

Para facilitar o desenvolvimento do sistema, é essencial ter em conta algumas das suas especificações.

Arquitetura

A arquitetura é um dos principais pilares para uma clara compreensão do sistema, uma vez que define a sua estrutura e estabelece os relacionamentos entre todas as suas partes constituintes.

Milestone 0
